    3. What is the variable used to facet the plot into multiple subplots?
    4. What is the scale function used to transform the y-axis into a log with base 10 scale?

    Solution

    1. x = carat
    2. The layer function is geom_hex.
    3. The variable color is used for faceting.
    4. The scale function for the y-axis is scale_y_log10.
    5. The label for the y-axis is “Price ($)”.
    6. The scale function for the x-axis is scale_x_sqrt.
    7. Notice here the legend is showing the fill color. So the scale function for the fill is scale_fill_viridis_c.
