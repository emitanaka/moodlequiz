    1. What is the theme argument name to change the facet label background to “indianred” color?
    2. What is the theme argument name to change the facet label text color to white?
    3. We want to change the panels so that the border is black and the background color is white. What is the name of the the theme argument name for this?
    4. What is the function name to modify the panel background?
    5. We want to modify the grid line colors from the default white to grey. What is the theme argument name for this?
    6. What is the function name to modify the look of the grid lines in a ggplot?
    7. What is the theme argument name to modify all axis titles?
    8. What is the function name to modify the look of axis labels in ggplot?
    9. What is the theme argument name to modify the look of the title text in ggplot?

    Solution

    1. strip.background
    2. strip.text
    3. panel.background
    4. element_rect
    5. panel.grid
    6. element_line
    7. axis.title
    8. element_text
    9. plot.title
